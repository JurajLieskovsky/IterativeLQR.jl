# IterativeLQR.jl
[![](https://img.shields.io/badge/docs-dev-blue.svg)](https://jurajlieskovsky.github.io/IterativeLQR.jl/dev/)

An implementation of the iterative linear-quadratic regulator (iLQR) with regularization by the way of minimal hessian modification. System's whose states lie on a manifold are supported through user supplied *state difference* and *coordinate Jacobian* functions.
